Beyond the Browser

There is life for JavaScript beyond the browser, and Node.js is a vehicle through which this is made possible. From AJAX calls to explicit DOM re-renders, enough APIs are exposed for a developer to dynamically bring data from the web and connect a client to it; but with the advent of Node.js as a run-time environment, JavaScript can now build the web, too. With a rapidly growing community providing a myriad of Node.js modules, full stack web applications can be engineered with JavaScript alone.

JavaScript Engines

The widely popular and open source JavaScript Engines1 are maintained by the major browsers in the market: Rhino (Mozilla Foundation), V8 (Google Chrome), and JavaScriptCore (Safari). Node.js runs on the V8 engine.

Node.js:

  • open-source
  • cross-platform runtime environment
  • single-threaded, event-driven Web servers
  • non-blocking I/O

Basics modules are written in JavaScript and compiled by the V8 engine.
